# QA-Testing-Portfolio

## About
This project demonstrates my QA testing approach on a web application, including test planning, test case design, and bug reporting.

## Objective
The goal was to validate core functionality, identify defects, and ensure a reliable user experience.

## Scope

Features Tested:
- Authentication (Login / Signup)
- Form Validation
- Navigation and Routing
- Core Application Functionality

Out of Scope:
- Performance testing
- Security testing (advanced)

## Tools
Postman – API testing
Jira (simulated) – bug tracking
Browser DevTools – debugging
SQL – database validation
